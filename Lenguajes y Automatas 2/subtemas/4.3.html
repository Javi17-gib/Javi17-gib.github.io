Volver al temario

¿Qué es el Lenguaje Máquina

Es el código binario directamente interpretable por el hardware. Cada instrucción tiene un formato específico que depende de la arquitectura.
El compilador debe generar este código de forma precisa para asegurar que el programa funcione correctamente


Tipos de Lenguaje Máquina


    Código binario puro:

    Instrucciones en forma de cadenas de 0 y 1.


    Código hexadecimal:

    Representación más compacta y legible del binario..


    Formato específico del procesador::

    Cada CPU tiene su propio formato de instrucciones binarias.
