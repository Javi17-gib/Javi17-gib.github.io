1.3 Comprobaciones de tipos en expresiones

Volver al temario

¿Qué es la comprobacion de tipo?

La comprobación de tipos es una verificación automática que asegura que las operaciones
en el código se realicen entre tipos de datos compatibles. Esta comprobación previene errores
lógicos que podrían pasar desapercibidos durante la etapa sintáctica. Si se detectan incompatibilidades,
el compilador debe reportarlas como errores semánticos.


Componentes de la comprobacion de tipo


    compatibilidad:

    Verifica si dos tipos de datos pueden usarse juntos (ej. int con float).


    Inferencia:

    El compilador adivina el tipo de una variable según su valor (ej. var x = 5;).


    Conversion:

    Cambia un tipo de dato a otro de forma explícita (ej. (int)3.5 se convierte en 3).


    Coerencia:

    El compilador convierte automáticamente un tipo a otro compatible (ej. sumar int con float convierte el int a float).
